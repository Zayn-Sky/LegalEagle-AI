# LegalEagle AI

LegalEagle AI 是一款面向中国法律咨询场景的智能问答助手。用户可以像向律师描述问题一样提问，系统会检索本地法规库，给出带法律依据的初步分析和行动建议。

它适合用于：

- 了解劳动争议、合同违约、民事责任等常见法律问题。
- 快速查找相关法律依据和条款。
- 获得一份结构清晰的初步咨询意见。
- 判断下一步应准备哪些事实和证据。

## 能帮你做什么

**直接回答法律问题**  
你可以直接输入问题，例如：

- “劳动者被拖欠工资怎么办？”
- “合同一方不履行合同义务应承担什么责任？”
- “公司没有按时支付劳动报酬，我可以怎么维权？”

**给出法律依据**  
回答会尽量引用具体法律名称和条款，并把核心结论对应到来源编号。

**整理处理思路**  
答案会按“结论、证据矩阵、法律依据、法律分析、行动建议、风险提示、需要补充的事实”组织，方便你快速判断下一步怎么做。

**先追问关键事实**  
如果缺少判断法律关系或责任结果所必需的事实，系统会先提示需要补充的信息。

**优先使用现行法规**  
系统默认优先使用现行有效法规，减少引用已废止依据的情况。

**支持网页使用**  
除了命令行，也提供一个极简网页问答页面，打开浏览器即可使用。

## 覆盖范围

`python scripts/scrape_npc_laws.py`  # 从国家法律法规数据库公开接口采集法规数据到本地

当前法规库覆盖：

- 民法商法：47 部
- 社会法：45 部

主要适用于劳动用工、社会保障、合同责任、民事权利义务、公司和商事相关问题的初步咨询。

## 使用方式

启动网页服务：

```bash
python -m legaleagle_agent.server --host 127.0.0.1 --port 8000
```

然后在浏览器中打开：

```text
http://127.0.0.1:8000
```

也可以使用命令行提问：

```bash
python -m legaleagle_agent.agent "劳动者被拖欠工资怎么办？"
```

## 本地部署

首次运行前需要安装依赖并配置模型服务：

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

在 `.env` 中填写 LLM 和 embedding 服务密钥后，构建本地法规索引：

```bash
python scripts/run_full_pipeline.py
```

## 重要提示

- LegalEagle AI 提供的是法律信息辅助和初步分析，不替代律师正式法律意见。
- 具体案件仍需要结合完整事实、证据材料、地方实践和专业律师判断。
- 如果问题事实不充分，系统会提示还需要补充哪些关键信息。
